# Customer Lifetime Value Analysis and Future Spending Prediction

## Project Objective
This project analyzes customer behavior and predicts future spending using machine learning techniques.

## Tasks Performed
- Business Problem Understanding
- Data Cleaning
- Feature Engineering
- Exploratory Data Analysis
- Customer Segmentation
- Future Spending Prediction
- Business Recommendations

## Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn

## Machine Learning Model
- Linear Regression

## Customer Segmentation
- KMeans Clustering

## Key Insights
- Previous spending strongly affects future spending
- High-value customers can be identified using segmentation
- Customer engagement positively impacts revenue

## Business Recommendations
- Improve customer retention
- Focus on high-value customers
- Reduce cancellation rates
- Increase customer engagement

## How to Run
1. Upload dataset to Google Colab
2. Run all notebook cells
3. Review outputs and visualizations
